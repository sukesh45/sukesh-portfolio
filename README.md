# Sukesh S - Portfolio

A modern, responsive personal portfolio website showcasing my skills, experience, and projects as a Full Stack Developer and IoT Enthusiast.

## 🚀 Live Demo

[View Portfolio](https://sukesh45.github.io/portfolio/) (if deployed)

## ✨ Features

- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Dark/Light Theme Toggle**: Switch between themes with smooth transitions
- **Interactive Sections**:
  - Hero section with animated blob
  - About section with personal details
  - Skills section with animated progress bars
  - Experience timeline with tabbed navigation
  - Projects showcase with hover effects
  - Contact form with validation
- **Smooth Animations**: CSS animations and JavaScript interactions
- **Mobile-First Navigation**: Hamburger menu for mobile devices
- **SEO Optimized**: Proper meta tags and semantic HTML

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Custom properties, flexbox, grid, animations
- **JavaScript (ES6+)**: DOM manipulation, event handling, theme switching
- **Font Awesome**: Icons for visual enhancement
- **Google Fonts**: Poppins and Playfair Display typography

## 📋 Sections

1. **Home**: Hero section with introduction and call-to-action buttons
2. **About**: Personal information, education, and contact details
3. **Skills**: Technical skills with animated progress bars
4. **Experience**: Timeline of internships, education, and achievements
5. **Projects**: Showcase of key projects with technologies used
6. **Contact**: Contact form and social media links

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Sukesh45/portfolio.git
   cd portfolio
   ```

2. Open `index.html` in your preferred web browser

No additional setup or dependencies required!
